# OWASP Juice Shop Security Assessment

## Overview

This repository contains a Web Application Penetration Testing (WAPT) report conducted against the OWASP Juice Shop application as part of an academic cybersecurity project.

## Objectives

- Perform security assessment of a web application
- Identify OWASP vulnerabilities
- Analyze security risks
- Provide remediation recommendations

## Tools Used

- Burp Suite
- Kali Linux
- OWASP Juice Shop
- Firefox
- FoxyProxy

## Skills Demonstrated

- Web Application Penetration Testing
- Vulnerability Assessment
- HTTP Request Analysis
- Security Reporting
- Risk Assessment
